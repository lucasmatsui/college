# Primeiro projeto Javascript

## Extençoes usadas:
- VScode icons
- Live server

## Comandos Terminal:
- cd = muda o diretorio.
- mkdir = cria um diretorio.
- ls/dir = lista o conteudo do diretorio.
- node --version = mostra a versão do node instalada no momento.
- code . = abre um diretorio no VSCode
